## Run Locally

Download the project

```bash
  git clone https://github.com/Warrikard/exo_cyberdian
```
open on vite.js

Install dependencies

```bash
  npm install
  npm add -D sass
  npm i vue-router
  npm run dev
```

Start the server

```bash
  npm run start
```
